
# Management Portal

## Two Modes

- **Admin Mode**
- **Manager Mode**

## Admin Mode

Login using Google login. Access is restricted to the following admin Google accounts:
- wwguyww@gmail.com
- ronenwes@gmail.com

### Admin Options

#### Create/Update/Delete Intercom System

Each intercom has:
- Name
- Address
- Manager Google account
- Status (active/disabled/deleted)

#### Perform Manager Options

Admins can perform manager options on behalf of managers.

## Manager Options

> **IMPORTANT:** Each manager can only work with intercom systems where they are defined as the manager.

### Add/Remove Intercom Residents

Each resident has:
- Apartment name
- Apartment number
- Google account
